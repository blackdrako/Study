Для чего нужно опорное решения - найти оптимальный план на основании фунд. теоремы.
Среди оптимальных планов задачи линейного
программирования в канонической форме обязательно есть опорное
решение ее системы ограничений. Если оптимальный план задачи
единственен, то он совпадает с некоторым опорным решением. 

В каком виде должна применятся к чистому виду
1.Канонический
2.Система огр.приведина к допустиму базовому виду.
3.Из целевой функции иск. базисные переменные.

Теорема 3.1. Для всякой пары двойственных задач если исходная
задача имеет решение, то двойственная задача также имеет
решение, и оптимальные значения целевых функций этих задач
совпадают, то есть zmax = fmin. Если исходная задача не имеет решения
по причине неограниченности целевой функции на области
допустимых значений, то двойственная задача недопустима.
